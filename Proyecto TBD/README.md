# Digital Barter – Fixed
Proyecto corregido con **React** + **Vite** + **Tailwind** + **React Router**.
- Login redirige a /home (simulación)
- Panel principal (Home) visible y sin imports rotos
- Botón "Crear una cuenta" visible

## Uso
```bash
npm install
npm run dev
```
Abrir la URL que imprime Vite (ej: http://localhost:5173/)
