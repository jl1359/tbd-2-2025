# Monorepo: UI + API + DB (React + Vite + Tailwind / Node + Express + Prisma / MySQL)

## Dev rápido
```bash
# en la raíz
npm install
npm run dev
```

## Requisitos
- Node 18+
- Docker (opcional para DB)

## Estructura
- `frontend/` React + Vite + Tailwind
- `backend/`  Node + Express + Prisma (MySQL)
- `infra/db/` SQL de init y seeds
